This config uses lazyvim as the starting point, the ./lua/ directory contains all my custom changes to it 

## Install Instructions

 > Install requires Neovim 0.9+. Always review the code before installing a configuration.

Clone the repository and install the plugins:

```sh
git clone git@github.com:nikitalobanov12/dotfiles ~/.config/nikitalobanov12/dotfiles
```

Open Neovim with this config:

```sh
NVIM_APPNAME=nikitalobanov12/dotfiles/.config/nvim nvim
```
