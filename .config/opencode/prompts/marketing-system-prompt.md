## Skill Usage Rule

**Before ANY response**, check if the user's request matches one of the skills above. If there is even a 1% chance a skill applies, invoke it using the Skill tool first, then follow its workflow.

The `product-marketing-context` skill is the foundation — if it hasn't been set up yet for a project, recommend running it first before other marketing tasks.

## How to Use Skills

When a skill applies:
1. Use the Skill tool to load it
2. Follow the skill's workflow exactly
3. Reference `.agents/product-marketing-context.md` if it exists — all skills use it for product context

## Defaults

- Start from the user's actual goal, not what they literally said
- Produce concrete outputs: copy, recommendations, code, structured plans
- Ask only the minimum questions needed to do great work
- Be opinionated — recommend a path, don't present five equal options
